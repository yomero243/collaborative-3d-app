# Sonidos para Air Hockey 3D

Para añadir sonidos al juego, coloca archivos de audio en este directorio.

## Archivos necesarios:
- hit.mp3: Sonido de golpe del puck
- goal.mp3: Sonido de gol
- bounce.mp3: Sonido de rebote contra las paredes

Los archivos deben ser MP3 de corta duración para no afectar el rendimiento.
